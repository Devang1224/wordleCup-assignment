## **How to install and run locally ?**

## clone the repository
```
$ git clone https://github.com/Devang1224/wordleCup-assignment.git
```
## Start the server
```
$ cd server
$ npm install
$ npm run start
```

## Start the application
```
$ cd client
$ npm install
$ npm run dev
```

## **Built with -**

- React JS
- Tailwind css



- Typescript
